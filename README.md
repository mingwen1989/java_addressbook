# Java Address Book

#### By _Ming Wen_

## Description

An address book that stores contact information for people you know. Stores first name, last name, birth month, etc. The user can select a contact, and add phone numbers. Each contact can store multiple phone numbers, and each phone number can store an area code, a number, and a type, like "home" or "cell". The user can add multiple emails and mailing addresses for each contact, along with a type (home, office, etc.) 

## Setup and Installation Requirements

  Clone the repo.  
  Run a instance of the gradle web server with 'gradle run'.  
  Navigate to localhost:4567

  Additionally, tests can be run with 'gradle test'.
